# Presentation Harness

발표 대본 작성/개선, reveal.js 슬라이드 제작, 주제 리서치, 청중 페르소나 피드백을 통합하여 최고의 발표 자료를 만드는 하네스. 새로 만들거나 기존 대본을 개선하는 두 가지 모드를 지원합니다.

## 에이전트 팀

| 에이전트 | 역할 |
|---------|------|
| researcher | 발표 주제에 대한 심층 리서치 |
| scriptwriter | 발표 대본 작성 (오프닝 훅 ~ 클로징) |
| slide-builder | reveal.js HTML 슬라이드 제작 |
| audience-reviewer | 청중 페르소나 기반 피드백 (agent-browser로 시각적 리뷰) |

## 워크플로우

```
신규: 리서치 → 대본 작성 + 슬라이드 제작 → 청중 피드백 → 개선 → 최종 산출물
개선: 기존 대본 분석 → 검증 리서치 → 대본 개선 → 슬라이드 제작 → 청중 피드백 → 최종 산출물
```

## 산출물

- `presentation-script.md` — 슬라이드별 발표 대본
- `presentation-slides.html` — reveal.js 단일 HTML 슬라이드 (브라우저에서 바로 열기)

## 사용법

```
발표 만들어줘: [주제], [시간], [청중]
```

예시:
```
"AI 도입 전략에 대해 20분짜리 발표를 만들어줘. 청중은 C레벨 경영진이야"
"마이크로서비스 아키텍처 소개 발표 15분, 백엔드 개발자 대상"
"이 대본 개선하고 슬라이드 만들어줘" (기존 대본 제공 시 자동으로 개선 모드)
```

## 선택적 의존성

```bash
# agent-browser (슬라이드 시각적 리뷰용, 선택)
npm install -g agent-browser && agent-browser install
```

설치하면 audience-reviewer가 실제 렌더링된 슬라이드를 브라우저에서 열어 시각적 피드백을 제공합니다. 미설치 시 HTML 소스 기반으로만 리뷰합니다.

## 슬라이드 보기

생성된 `presentation-slides.html`을 브라우저에서 열면 됩니다.
- 방향키: 슬라이드 이동
- `S` 키: 발표자 노트 (별도 창)
- `F` 키: 전체 화면
- `ESC`: 슬라이드 오버뷰
