# Market Analysis — 시장 분석

트렌드와 아이디어의 사업화 가능성을 분석하고, 수익성 있는 기회를 발굴하는 스킬.

## 분석 프레임워크

### Step 1: 타겟 세그먼트 분석

WebSearch로 다음을 조사한다:

| 분석 항목 | 조사 포인트 |
|----------|-----------|
| 인구통계 | 연령, 성별, 소득, 지역, 가구 구성 |
| 심리통계 | 가치관, 관심사, 소비 성향, 미디어 소비 패턴 |
| 행동 패턴 | 구매 채널, 의사결정 과정, 사용 빈도, 지불 의향 |
| 고통점 | 현재 해결되지 않는 문제, 불만족 요소, 비효율 |

**페르소나 구체화**: "MZ세대"처럼 넓은 세그먼트가 아니라, "서울 거주 28세 1인 가구 직장인, 월 가처분소득 150만원, 퇴근 후 넷플릭스 시청, 주말 카페 탐방" 수준으로 구체화한다.

### Step 2: 시장 규모 추정

가능한 한 실제 데이터를 수집하고, 없으면 bottom-up 추정한다:

- **TAM** (Total Addressable Market): 전체 잠재 시장
- **SAM** (Serviceable Available Market): 도달 가능한 시장
- **SOM** (Serviceable Obtainable Market): 현실적으로 획득 가능한 시장

추정 시 근거를 명시한다: "국내 1인 가구 750만 × 월 식비 평균 40만원 × 건강 간식 비율 5% = TAM 약 4,500억원"

### Step 3: 경쟁 환경 분석

| 분석 항목 | 내용 |
|----------|------|
| 직접 경쟁 | 동일 문제를 동일 방식으로 해결하는 기존 플레이어 |
| 간접 경쟁 | 동일 문제를 다른 방식으로 해결하는 대체재 |
| 잠재 경쟁 | 진입 가능성이 있는 대형 플레이어 |
| 차별화 기회 | 기존 플레이어가 놓치고 있는 지점 |

### Step 4: 수익 모델 탐색

최소 2가지 이상의 수익 모델을 비교한다:

| 모델 | 적합한 경우 | 핵심 지표 |
|------|-----------|----------|
| 구독 (SaaS) | 반복 사용, 높은 전환 비용 | MRR, Churn Rate |
| 프리미엄 | 넓은 사용자 기반, 명확한 상위 가치 | 전환율, ARPU |
| 커머스 | 물리적 제품, 반복 구매 | 객단가, 재구매율 |
| 광고 | 대규모 트래픽, 높은 체류 시간 | DAU, 체류 시간 |
| 수수료/중개 | 양면 시장, 거래 촉진 | GMV, Take Rate |
| 라이선스/B2B | 기업 고객, 전문 솔루션 | ACV, 계약 기간 |

### Step 5: 타이밍 분석

"왜 지금이 적기인가?"에 답한다:

- 기술 성숙도 (인프라가 준비되었는가?)
- 규제 환경 (규제가 열리고 있는가? 닫히고 있는가?)
- 소비자 행동 변화 (임계점을 넘었는가?)
- 경쟁 공백 (아직 지배적 플레이어가 없는가?)
